Manipulating the DOM

Finding Elements

  • document.getElementById(id)

    Fetches a single element by it's id attribute.

  • document.querySelector(selector)
    element.querySelector(selector)

    Finds the first match for selector or null if not found.

  • document.querySelectorAll(selector)
    element.querySelectorAll(selector)

    Finds all matches for selector or an empty NodeList if not found.

  • document.getElementsByClassName(class)

    Use document.querySelector('.class') , it's easier

    Not frequently used: Get all elements in the page by their CSS class.

  • document.getElementsByTagName(tag)

    Use document.querySelector('tag') , it's easier

    Not frequently used: Get all elements in the page by their tag .

Creating elements

  • document.createElement(elementName)

    Have to add the element to the DOM

  • element.insertAdjacentHTML(position, HTML)

    location can be 'beforebegin', 'afterbegin', 'beforeend', or 'afterend'
    Faster than innerHTML , below

  • element.insertAdjacentElement(position, newElement)

    location can be 'beforebegin', 'afterbegin', 'beforeend', or 'afterend'

  • element.innerHTML = 'Some HTML'

    Sets the HTML of a node, somewhat slow

  • node.cloneNode(deep)

    Copy this node. Shallow, unless deep is true

Text content

  • element.textContent = content

    Preferred. More efficient than innerText .
    See also textContent vs innerText

  • element.innerText = text

    Aware of styling, will not show hidden text/content.

Adding or moving elements

Given some DOM element el and a target t

  • t.appendChild(el) el is now the last child of t; inflexible
  • t.append(el) el is now the last child of t
  • t.append(...elements) each element is added as the last child of t
  • t.append(DOMString) add DOMString as text as the last child of t
  • t.append(...DOMStrings) each DOMString is added as text as the last child of t

  • With the same arguments as append()

    • prepend / first child of t
    • before / added before t as a sibling
    • after / added after t as a sibling

Removing elements

  • element.remove()

    The element removes itself from its parent

  • element.replaceWith(...nodes)

    The element is replaced with the provided node(s)

  • node.replaceChild(target, replacement)

    In node, replaces target with replacement

  • node.replaceChildren(...replacement)

    In node, replaces the child nodes of target with replacement

  • node.removeChild(target)

    In node, removes target
